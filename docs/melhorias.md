# Melhorias 

Esse documento tem como objetivo listar as melhorias que podem ser feitas no projeto, seja para corrigir bugs, otimizar o código ou adicionar novas funcionalidades.

## Correção de Bugs
- [ ] Corrigir o problema de cadastro do mesmo usuário mais de uma vez, garantindo que o sistema não permita duplicatas.

## Melhoriias 
- [ ] Adicionar testes unitários para garantir a qualidade do código.
- [ ] Refatorar o código para melhorar a legibilidade e manutenção.
- [ ] Melhorar espaçamento entre impressão dos elementos para facilitar melhor visualização.
